# 🎓 TracKOM - Tracing Kelulusan Mahasiswa

[![Laravel Version](https://img.shields.io/badge/Laravel-v10.x-FF2D20?style=flat&logo=laravel)](https://laravel.com)
[![PHP Version](https://img.shields.io/badge/PHP-v8.1+-777BB1?style=flat&logo=php)](https://www.php.net)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**TracKOM** adalah sistem informasi manajemen alumni yang dirancang untuk melacak data kelulusan, status pekerjaan, dan persebaran alumni perguruan tinggi secara efisien. Platform ini membantu institusi dalam mengumpulkan data *Tracer Study* untuk keperluan akreditasi dan pengembangan kurikulum.

---

## 🚀 Fitur Utama

- **📊 Dashboard Analitik:** Visualisasi data statistik lulusan dan status kebekerjaan secara real-time.
- **📂 Manajemen Data Alumni:** Pengelolaan biodata lengkap, riwayat pendidikan, dan pekerjaan.
- **📋 Kuesioner Tracer Study:** Sistem pengisian survei mandiri bagi alumni dengan antarmuka yang user-friendly.
- **🔍 Pencarian & Filter Canggih:** Memudahkan pencarian data berdasarkan tahun angkatan, program studi, hingga lokasi kerja.
- **📄 Laporan Ekspor:** Fitur export data ke format Excel/PDF untuk pelaporan birokrasi kampus.

---

## 🛠️ Stack Teknologi

- **Framework:** [Laravel 10](https://laravel.com/)
- **Frontend:** Blade Templating, Tailwind CSS / Bootstrap
- **Database:** MySQL / MariaDB
- **Tools:** Composer, NPM
