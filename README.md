# 📱 TracKOM - Lost & Found App

[![Android Studio](https://img.shields.io/badge/IDE-Android%20Studio-3DDC84?style=flat&logo=android-studio&logoColor=white)](https://developer.android.com/studio)
[![Firebase](https://img.shields.io/badge/Backend-Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)](https://firebase.google.com/)
[![Kotlin](https://img.shields.io/badge/Language-Kotlin%20/%20Java-007324?style=flat&logo=kotlin&logoColor=white)](https://kotlinlang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**TracKOM** adalah aplikasi mobile berbasis Android yang berfungsi sebagai platform sosial untuk membantu pengguna menemukan barang yang hilang atau melaporkan barang yang ditemukan. Dengan integrasi real-time, aplikasi ini memudahkan komunitas untuk saling membantu dalam mengembalikan barang kepada pemiliknya.

---

## ✨ Fitur Utama

- **🔍 Lapor Barang Hilang:** Pengguna dapat mengunggah detail barang yang hilang beserta foto dan lokasi terakhir.
- **🙋 Lapor Penemuan Barang:** Fitur untuk melaporkan barang yang ditemukan agar pemilik dapat mengenalinya.
- **🔥 Real-time Database:** Sinkronisasi data laporan secara instan menggunakan Firebase.
- **📸 Upload Foto:** Integrasi dengan Firebase Storage untuk mengunggah bukti foto barang.
- **💬 Fitur Kontak:** Memudahkan penemu dan pemilik untuk berkomunikasi.

---

## 🛠️ Stack Teknologi

- **Tools:** [Android Studio](https://developer.android.com/studio)
- **Language:** Kotlin / Java
- **Database:** Firebase Realtime Database / Firestore
- **Authentication:** Firebase Auth
